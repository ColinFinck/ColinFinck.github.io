ReactOS Website Translation Process

by frik85 | December 13, 2005

ReactOS Website Translation Process - Community Challenge

I started the ReactOS Website Translation Process one week ago. My plan was to start it with an information in the ReactOS IRC Web Channel (#reactos-web). The idea was to allow a small group of translators to translate ReactOS web contents with the RosCMS interface. Now, some interface bugs has been fixed and a lot of translated web contents has been added to the database (thanks to all contributors/translators).

The Translation Interface is now useavble and bug free. It's time to launch the translation process for everyone who want to constribute.

Website Translation Status page

If you want to help us to translate the Website to other languages, please read the Website Translation Tutorial!

The ReactOS Website is available in the following languages:

  • English (original content)
  • German
  • French
  • Spanish
  • Italian
  • Hungarian
  • Swedish

If you miss your mother tongue and you want to help to translate web contents, then please read the website translation tutorial and contact a web team member or a language maintainer.

Klemens Friedl <frik85>
