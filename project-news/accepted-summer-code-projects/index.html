Accepted Summer of Code Projects

by Z98 | April 26, 2011

The ReactOS project has accepted six entries from the submitted projects, covering a wide range of functionality. The completion of some would have immediate, user visible effects on ReactOS, while others are more long term.

TCP/IP Driver

Rewrite of the current TCP/IP stack using the lwIP library.

  • Student: Claudiu Mihail
  • Mentor: Art Yerkes

Completion of Explorer_New

Complete the implementation of a new explorer shell for ReactOS.

  • Student: Andrew Green
  • Mentor: Aleksey Bragin

Theme Support

Implement the theme service for the user interface.

  • Student: Giannis Adamopoulos
  • Mentor: Amine Khaldi

Audio Stream Mixing

Implement support for the mixing of audio streams for better sound controls.

  • Student: Neeraj Yadav
  • Mentor: Johannes Anderwald

Kernelmode Testsuite

Implement a testing system for the kernelmode ReactOS components to increase robustness and regression testing.

  • Student: Thomas Faber
  • Mentor: Pierre Schweitzer

GDI Font Driver

Implement a font driver to correctly render text through the Win32 subsystem.

  • Student: Timo Kreuzer
  • Mentor: Ged Murphy

We wish all of the participants the best of luck with their projects as each success will help bring ReactOS that much closer to a stable version.
