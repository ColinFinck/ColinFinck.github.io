Happy New Year!

by frik85

January 1, 2006

Happy New Year! 2005 was a great year for ReactOS. The following facts came from the SVN server (thanks to Casper Hornstrup): The core ReactOS code base is now 120MB large. The repository is currently 624MB large and contains ~20500 revisions. ~8000 of these revisions were created in 2005. This mean 39% of the total number of revisions created during ReactOS' entire 8 year history was created in 2005!
